Calendar Table Style Sample Pack

Each sample uses only inline styles (Roll20 compatible). Open in browser to compare.

Roll20 Font Support: Roll20 chat uses the page's default font stack. You can set font-family via inline styles, but only web-safe / system fonts work (no @font-face or Google Fonts in chat). Safe choices: Georgia, Palatino, "Trebuchet MS", Verdana, "Segoe UI", Tahoma, monospace.

Buttons: Always Roll20's default pink/magenta. Uses markdown syntax [Label](!cal cmd). The script cannot style buttons — Roll20 controls their appearance entirely. /direct broadcast mode strips buttons; only GM whispers show them.
