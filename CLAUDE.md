# Eberron Calendar — Roll20 API Script

A Roll20 API script (`calendar.js`) for managing an Eberron campaign calendar. Single file, ~13,000 lines. All state stored in Roll20's `state` object.

**Primary canon source**: Keith Baker's published Eberron material. Lore fidelity is a hard constraint.

## Key Files

- **`DESIGN.md`** — Authoritative design document. Architecture, mechanical systems, data tables, decision log. Read the relevant section before implementing anything.
- **`Design Tasks.md`** — Development workflow, task lifecycle, and implementation backlog. All work is tracked here.
- **`design/`** — Active design discussions (undecided topics). Do not implement anything found only here.
